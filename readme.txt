1、index.html是增加了js即能正常跑的页面。
2、index-static.html 是在最初始做效果的静态页面，没有增加任何js.
3、function.css是常用样式文件。
4、index-page.css 是页面模块及布局样式文件。
5、media.css是响应式样式文件。
6、page-unit.css是页面中元件样式文件。
7、reset.css是全局的reset样式文件。
8、js/base是基础类js包。
9、js/module是页面模块js化包。
10、js/util是工具类js包
11、设计文件夹是一些设计类导出图片，由于设计软件没有注册，在导出图片时有水印。
12、sprites.psd是页面部分常用图片的集合psd文件。
备注，本实现没有采用任务js模块依赖，所用依赖都是通过自己在程序中手工控制，
所以请不要调整js加载顺序。
